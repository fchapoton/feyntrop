feyntrop
========

Building:

* cd extern
* git clone https://github.com/pybind/pybind11.git
* git clone git@gitlab.com:libeigen/eigen.git
* cd ..
* make feyntrop.so
* mv feyntrop.so tutorials_and_examples

Run the tutorial:

* cd tutorials_and_examples
* jupyter notebook
* run the tutorial_2L_3pt.ipynb in browser
